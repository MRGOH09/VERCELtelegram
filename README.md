## 🚀 快速开始

1. 发送 `/start` 完成初始设置
2. 发送 `/record` 记录一笔支出
3. 发送 `/my` 查看个人统计（默认本月）
4. 发送 `/history` 查看历史记录（默认本月）
5. 发送 `/settings` 修改资料与预算

## 🔧 命令说明

- **`/start`** - 初始设置（注册/修改资料）
- **`/record`** - 记录一笔支出
- **`/my`** - 查看个人统计（默认本月，可切换时间段）
- **`/history`** - 查看历史记录（默认本月，可切换时间段）
- **`/settings`** - 修改资料与预算
- **`/help`** - 功能指南
- **`/edit`** - 编辑记录（格式：edit ID 金额 备注）
- **`/delete`** - 删除记录（格式：delete ID）
- **`/broadcast`** - 发送广播消息（仅管理员）
