# 启用Google Apps Script API指南

## 🚨 需要先启用API

在继续之前，你需要启用Google Apps Script API：

## 📋 步骤：

### 1. 点击以下链接启用API：
🔗 **https://script.google.com/home/usersettings**

### 2. 在打开的页面中：
- 找到"Google Apps Script API" 
- 将开关设置为"开启"(ON)

### 3. 等待几分钟让设置生效

### 4. 回到终端继续部署：
```bash
# 重新运行部署脚本
./deploy.sh
```

## 🎯 完成后：

启用API后，clasp就能：
- ✅ 创建新的GAS项目
- ✅ 上传你的脚本文件
- ✅ 自动打开GAS编辑器

## 💡 一次性设置：

这个设置只需要做一次，以后所有clasp操作都会正常工作。
